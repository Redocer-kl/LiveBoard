# LiveBoard - Real-time Collaborative Whiteboard

A web application for real-time collaborative drawing, demonstrating the integration of WebSockets with a modern tech stack.

## Current State (WIP)
- **WebSocket Connectivity**: Functional real-time communication between the client and server.
- **Frontend**: Interactive drawing canvas built with React.
- **Backend**: Django Channels architecture (infrastructure for real-time updates).

## Roadmap
- [ ] Persistence: Saving boards to PostgreSQL.
- [ ] Multi-room support for different collaborative sessions.

## Tech Stack
- **Frontend**: React 18, WebSockets.
- **Backend**: Python, Django, Django Channels.
